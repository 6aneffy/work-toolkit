# Work Toolkit (업무 입력 훈련 워크북)

## 프로젝트 개요

엑셀과 유사한 인터페이스에서 업무형 데이터를 입력하며 타자 정확도를 훈련하는 웹 애플리케이션을 구현한다.

또한, 단축키를 통해 간단한 업무 내용을 보고용 문장으로 변환해주는 보조 기능을 제공한다.

이 프로젝트는 **게임이 아닌 업무형 입력 훈련 도구**이다.

---

## 기술 스택

- React
- TypeScript
- Vite
- 상태관리: React state 또는 Zustand
- 스타일: Tailwind 또는 CSS Modules

## 핵심 기능

### 1. 엑셀 스타일 입력 훈련

- 엑셀처럼 보이는 그리드 UI
- 샘플 데이터를 보고 그대로 입력
- 키보드 중심 인터랙션

#### 키보드 규칙

- Tab → 오른쪽 셀 이동
- Shift + Tab → 왼쪽 셀 이동
- Enter → 다음 행 첫 셀 이동
- 방향키 이동 (가능하면 구현)
- 기본 Tab 포커스 이동은 막는다

---

### 2. 입력 방식

- 현재 활성 셀만 입력 가능
- 입력 완료 시 정답과 비교
- 틀리면 셀을 빨간색으로 표시
- 수정 후 맞으면 정상 상태로 복구

---

### 3. 시간 및 정확도

표시 정보:

- 셀 위치 (예: B3)
- 전체 경과시간 (mm:ss)
- 현재 행 입력시간 (x.x초)
- 정확도 (%)

동작 규칙:

- 첫 입력 시 전체 시간 시작
- 행 이동 시 해당 행 시간 리셋

---

### 4. 문장 변환 모달

#### 실행

- 단축키: Ctrl + /

#### 특징

- 메인 입력 기능과 연결되지 않음
- 독립적인 보조 기능

#### UI 구성

- 제목: 업무 문장 변환
- 입력창 1개
- 결과 출력 영역 1개
- 버튼:
  - 복사
  - 다른 결과 보기
  - 닫기

#### 동작

- 사용자가 직접 문장을 입력
- 결과는 1개만 표시
- "다른 결과 보기" 클릭 시 다른 템플릿 출력
- 복사 버튼 → 클립보드 복사
- ESC → 모달 닫기

## 아키텍처 요구사항

- 비즈니스 로직과 UI를 분리한다
- 데이터 처리 로직은 utils로 분리한다
- 문장 변환 로직은 별도 모듈로 분리한다
- localStorage는 wrapper 함수로 접근한다
- 모든 주요 데이터는 TypeScript 타입으로 정의한다

## 문장 변환 템플릿 예시

"서류 정리" ->
- 문서 분류 및 정리 업무 수행
- 관련 서류를 검토하고 정리함
- 자료 정리 및 문서 관리 업무를 지원함

"자료 조사" ->
- 관련 자료를 조사하고 정리함
- 업무 관련 자료 조사 및 기초 검토를 수행함
- 필요한 자료를 수집하고 내용을 정리함

"회의 참석" ->
- 회의에 참석하여 주요 내용을 파악함
- 회의 내용 청취 및 주요 사항을 정리함
- 관련 회의에 참여하여 업무 흐름을 파악함

"엑셀 입력" ->
- 데이터 입력 및 기초 검토 작업 수행
- 엑셀 기반 자료 입력 업무를 지원함
- 관련 데이터를 입력하고 정리함

"문서 확인" ->
- 문서 내용을 검토하고 정리함
- 관련 문서를 확인하여 필요한 사항을 점검함
- 문서 검토 및 기초 확인 업무를 수행함

"영상 제작" ->
- 영상 제작 업무를 수행함
- 관련 영상 콘텐츠를 제작하고 정리함
- 업무 목적에 맞는 영상을 제작하여 활용함

키워드가 명확하지 않을 경우 공통 템플릿 사용:

- 관련 업무를 수행하고 내용을 정리함
- 담당 업무를 지원하며 필요한 사항을 확인함
- 업무 진행 과정에서 관련 자료를 검토하고 정리함
